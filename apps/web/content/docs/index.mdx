# Overview

DEC Bench is an open-source benchmark for evaluating AI agent competency on real-world data engineering tasks. It measures performance against a real infrastructure stack (Postgres, Redpanda, ClickHouse) and scores agents on correctness, latency, cost, quality, and efficiency.
