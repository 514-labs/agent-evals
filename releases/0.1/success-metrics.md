## Tracking Setup

### GitHub
- Watch repo insights for star history and traffic
- Monitor release asset download counts for CLI installs
- Track forks as a secondary engagement signal

### Vercel Analytics
- Already enabled on the web app
- Track `/`, `/leaderboard`, `/audit/*`, `/docs/*` pageviews
- Watch referrer breakdown to see which channels drive traffic

### HN
- Check placement at 1h, 2h, 4h, 8h, 24h after posting
- Count total comments and upvotes at 24h and 48h

### Manual
- Search Twitter for "DEC Bench" and "dec-bench" daily for the first week
- Search Reddit for cross-posts or mentions outside the launch threads
- Check Data Engineering Weekly, TLDR, and other newsletters for pickups
